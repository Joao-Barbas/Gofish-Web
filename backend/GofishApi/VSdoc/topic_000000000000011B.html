AppException Class

Namespace: GofishApi.Exceptions
Assembly: GofishApi (in GofishApi.dll)
Base exception for application-level errors with settable HTTP status codes. This is intended to follow the machine-readable format for specifying errors in HTTP API responses based on https://tools.ietf.org/html/rfc7807.
